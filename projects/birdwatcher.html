BIRDWATCHER

Application d'observation et d'identification d'oiseaux utilisant la vision par ordinateur et l'intelligence artificielle pour la détection en temps réel.

CODE SOURCE (v1) CODE SOURCE (v2)

Description

Birdwatcher est un projet de vision par ordinateur dédié à l'identification automatique d'espèces d'oiseaux. Grâce à des algorithmes de machine learning, l'application peut détecter et classifier les oiseaux en temps réel.

Le projet a évolué en deux versions : la v1 en Python pur avec des modèles de base, et la v2 avec une interface web HTML améliorée et des modèles plus performants.

Fonctionnalités clés

  • 01 Détection d'oiseaux en temps réel par caméra
  • 02 Classification automatique des espèces
  • 03 Base de données d'espèces avec informations détaillées
  • 04 Interface web pour visualiser les résultats (v2)
  • 05 Modèles ML entraînés et optimisés
