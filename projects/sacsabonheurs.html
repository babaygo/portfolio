Description

Sacs à Bonheurs est une application e-commerce moderne construite avec Next.js et Express, offrant une expérience d'achat complète avec intégration Stripe, dashboard admin et livraison Mondial Relay. L'architecture JAMstack sépare clairement le frontend (Vercel) du backend (Render).

Le site propose un catalogue de sacs artisanaux avec filtrage par catégorie, un panier persistant via Zustand, un checkout sécurisé par webhooks Stripe, et un blog intégré avec éditeur rich text TipTap. L'admin gère les produits, commandes, contenus légaux et bannières.

Fonctionnalités clés

  • 01 Catalogue produits avec filtrage par catégorie et fiches détaillées
  • 02 Paiement sécurisé Stripe avec création de commande par webhook
  • 03 Livraison Mondial Relay — sélection de point relais via widget
  • 04 Dashboard admin : CRUD produits, catégories, commandes, bannières
  • 05 Authentification sécurisée (better-auth) avec contrôle d'accès 3 niveaux
  • 06 Blog intégré avec éditeur TipTap, SEO SSR et images Cloudflare R2
